A CTA WordPress plugin for RockContent Technical Challenge Job Interview

== Description ==
A CTA WordPress plugin for RockContent Technical Challenge Job Interview. You can create banners with or without a link, using shortcodes to put them on page and posts.


== Installation ==
1. Upload "test-plugin.php" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
